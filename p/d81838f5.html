修改提示:

最后几行:

var sw=window.screen.width;
if (sw>1200) {
    POWERMODE.colorful=true; 
    POWERMODE.shake=true;
    document.body.addEventListener('input',POWERMODE)
}

中POWERMODE.colorful=true;指是否开启多彩颜色。而POWERMODE.shake=true;指是否开启屏幕震荡.
默认全部开启.
